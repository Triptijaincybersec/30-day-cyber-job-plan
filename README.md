# 30 Day Cyber Job Plan

**Live Dashboard:** https://triptijaincybersec.github.io/30-day-cyber-job-plan/

**What this is:** Public 30-day learning tracker for SOC Analyst role. Built with Python + GitHub Pages. Auto-updates on every commit.

**Schedule:** Monday to Sunday. Sunday = Rest + Review day with lighter tasks. 6:00 AM - 6:00 PM IST practice slots.

**Stack:** Python, HTML/CSS, GitHub Actions, Kali Linux, pfSense, Splunk, AWS, Azure, Wireshark

**Progress:** 30 Days Total | 4 Weeks + 2 Days

**Structure:**
- **Week 1:** Setup + CCNA Basics + Linux
- **Week 2:** Firewall + SIEM Intro + MITRE
- **Week 3:** IR Process + Cloud Basics + AWS/Azure
- **Week 4:** Interview Prep + Portfolio + Applications
- **Week 5:** Networking + Final Ship

**How it works:** 
1. Edit `progress.json` → mark tasks `"done": true`
2. Run `python update.py` → generates dashboard
3. Push to GitHub → Site updates automatically
