\## Day 2 SOC Notes

1\. Tool: SOC analysts use SIEM tools like Splunk or Microsoft Sentinel to monitor logs and detect suspicious network activity

2\. Tickets: Alerts come into a queue like ServiceNow. L1 analysts triage first, check false positives, escalate P1/P2 to L2

3\. Surprised by: Job is 70% investigation + documentation. Lots of report writing, alert fatigue is real



\## Day 2 Research: 3 Cyber Roles

1\. SOC L1: Splunk + investigate phishing alerts \& log analysis

2\. IT Support: ServiceNow + user account issues \& basic troubleshooting

3\. NOC Engineer: SolarWinds + monitor network uptime \& escalate outages
